## Features:

- Login & Register User
- Add & delete a Book
- Upload file to Vercel blob

## Technology Stack:

- Next.js
- Next Auth
- Prisma
- TypeScript
- TailwindCSS
- Shadcn UI

## Usage

### Env Variables

Create a .env file in then root and add the following

```
POSTGRES_URL=""
POSTGRES_PRISMA_URL=""
POSTGRES_URL_NON_POOLING=""
POSTGRES_USER=""
POSTGRES_HOST=""
POSTGRES_PASSWORD=""
POSTGRES_DATABASE=""

BLOB_READ_WRITE_TOKEN=""

NEXTAUTH_URL=
NEXTAUTH_SECRET=
```

## Install Dependencies

```
npm install
```

### Run

```
# Run project
npm run dev
```
